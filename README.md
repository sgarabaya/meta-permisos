# Meta: Permisos

Este es el repositorio para un proyecto para Organizacion Empresarial en la Escuela Da Vinci.

Desarrollo de una plataforma que permita a los usuarios comprender y gestionar mejor el uso de sus datos personales en redes sociales.
Ofrecería herramientas para visualizar cómo se usan sus datos y configurar permisos de manera más sencilla e interactiva.

## Instrucciones:

1. Instalar NodeJS (asegurarse que los comandos `node` y `npm` sean accesibles desde la consola)
2. Abrir la consola en la carpeta del proyecto
3. Correr `npm install`, esto va a instalar las dependencias (Principalmente `vite`)
4. Luego, correr `npm run dev`, esto va a iniciar "dev mode" y abrir una pestania del navegador.


Una vez que `dev mode` esta iniciado, esto permite editar los archivos html, js y css, y al guardar la pagina se actualiza con los cambios en tiempo real.
